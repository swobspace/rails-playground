= rails-playground

Rails playground for testing stimulus, javascript and new rails features.

Using:

* rails 7.0
* jssbundling-rails with esbuild
* cssbundling-rails with bootstrap 5

== HowTo

.generate app
[source,sh]
----
rails new rails-playground \
  --database=sqlite3 \
  --skip-test \
  --skip-jbuilder \
  --javascript esbuild \
  --css bootstrap
----
